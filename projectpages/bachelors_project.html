Bachelor’s Thesis

Auditory and Visual feedback: Impact on player performance and comprehension in First Person Shooters

Overview

In this project I conducted a user study together with Alexander Gemal in order to try and compare the impacts of visual and auditory feedback in FPS games and how they impact player performance. This consisted of creating a small FPS game in unity acting as our test environment and then conducting the user tests.

Tech Stack

UnityC#

Features

  • Created a small test environment in Unity using a template
  • Conducted a user study utilizing test environment
  • Analyzed results using both qualitative and quantitative methods

My Role & Challenges

In the start of the project I worked a lot on getting our test game in Unity working according to our specifications. For me this mostly included working on the level design for our test levels. Furthermore the initial phase of the thesis involved a lot of research into user interfaces (UI) and Audio in video games and their role in the player experience. The rest of the time was spent on conducting user studies and analyzing the results and then composing the final report.
