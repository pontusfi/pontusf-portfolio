EA SEED — Hierarchical AI for Smarter NPCs

Combining the strategic strengths of LLMs with the fast reflexes of RL to create smarter, more adaptive NPCs.

Overview

The goal of this project is to develop intelligent, adaptive NPCs for games by combining Reinforcement Learning (RL) with Large Language Models (LLMs) in a hierarchical system. RL agents handle fast, low-level behaviors such as navigation and combat, while the LLM oversees strategic decision-making and orchestrates which RL agents to activate based on the game state.

Features (In Progress)

  • Hierarchical AI Control: Combining an LLM for strategy with RL agents for fast, low-level actions.
  • Adaptive NPC Behavior: NPCs can switch between sub-behaviors (e.g. defensive, aggressive) based on game state.
  • Smarter & Faster Gameplay: Balances long-term planning with real-time responsiveness for more engaging NPCs.

My Role & Challenges (In Progress)

In Progress: In this EA SEED project, my primary role is setting up the training environment in Unity and preparing RL agents for training. The next steps involve developing sub-behaviours (defensive, aggressive) using RL and connecting them to an LLM controller for strategic decision-making.
