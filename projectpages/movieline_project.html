Movieline

Timeline game with movies

Overview

A web based interactive timeline game with the goal of placing as many movies as you can in the correct order. We developed this project using React with Vite and utilized Firebase in order the handle user data.

Screenshots

Tech Stack

HTMLCSSReact

Features

  • Created a timeline based game with movies.
  • Used OMDB API to fetch movie data.
  • Built website with React following the MVP (Model-View-Presenter) architecture.
  • Saving user data in database created with Firebase.

My Role & Challenges

During this project, I focused heavily on developing the Presenters and Views within the MVP architecture, ensuring a clean separation of concerns and maintainable code. I also implemented much of the routing logic and configured the React Router setup for seamless navigation. In addition, I was responsible for establishing data persistence using Firebase. This included configuring the backend and implementing functionality to store and retrieve data from the database.
